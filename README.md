# MuZap

MuZap — Windows-менеджер для запуска и установки в автозагрузку стратегий **zapret winws** (DPI bypass) с удобным меню, обновлением списков и диагностикой.

Проект ориентирован на практичность:
- поставить стратегию как Windows Service;
- быстро переключать режимы (Game Filter / IPSet Filter / Auto-update check);
- обновлять ipset-лист и hosts-блок (для некоторых сценариев);
- запускать диагностику конфликтов и тесты стратегий.

---

### Почему антивирус может ругаться
MuZap использует WinDivert-драйвер (перехват/фильтрация пакетов). Такие компоненты часто попадают в категории RiskTool/PUA/“hacktool”, потому что **могут использоваться и легитимными, и вредоносными программами**.

## Быстрый старт (Windows)

1) Скачайте архив со страницы Releases.

2) Если Windows показывает кнопку **“Разблокировать”** в свойствах файла (для скачанного из интернета архива) — включите её и нажмите OK.

3) Распакуйте архив в папку без кириллицы/спецсимволов (желательно что-то вроде `C:\MuZap\`).

4) Запустите `MuZap.bat`.
- Скрипт запросит права администратора (UAC) и перезапустит сам себя с админ-правами.

5) В меню:
- **Install Service** — выбрать стратегию из `strategies.ini` и установить как Windows Service.
- **Restart Service / Remove Services** — управление сервисом.
- **Update IPSet List / Update Hosts File** — обновления данных.
- **Run Diagnostics** — диагностика конфликтов и типовых проблем.
- **Run Tests** — тест прогон стратегий.

## Файлы и структура

- `MuZap.bat` — основной менеджер (меню, установка сервиса, диагностика).
- `muzap.ini` — пользовательские настройки:
  - `CheckUpdates` (0/1)
  - `GameFilterMode` (off/all/tcp/udp)
  - `Hosts.BackupMode` (off/once/single/timestamp)
- `strategies.ini` — набор стратегий (секции `[NAME]`, поля `Description` и `Params`).
- `lists/` — доменные и IP-списки:
  - `list-general.txt`, `list-google.txt`, `list-exclude.txt`
  - `ipset-all.txt`, `ipset-exclude.txt`
  - при первом запуске создаются `*-user.txt` (пользовательские добавления/исключения)
- `utils/hosts_manage.ps1` — безопасное обновление/удаление блока MuZap в system hosts с бэкапами.
- `utils/test_muzap.ps1` — тесты стратегий (стандартные и DPI-checkers).
- `bin/` — бинарники (winws.exe, WinDivert, и др.)

## Обновления

### Проверка обновлений (встроенная)
В меню есть пункт **Check for Updates**.
Он сравнивает вашу локальную версию с версией в репозитории и открывает страницу последнего релиза.

Если проверка обновлений не нужна — отключите:
- Menu → **Auto-Update Check**
